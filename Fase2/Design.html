Proposito del sitio

El proposito de este sitio sera el de servir como una guia base en el desarrollo web a traves del uso de las tecnologias de Flex Box y CSS Grid identificando sus diferencias, sus herramientas y sus aplicaciones en el desarrollo web. En Este curso se abordaran las tematicas de CSS Grid con el cual aprenderas a ubicar elementos en dos dimensiones, columnas y filas al tiempo, mientras que con Flexbox aprenderas a manejar una dimensión, donde tienes que elegir entre filas y columnas.
