Flexbox o CSS Grid


Desde que se lanzo CSS Grid por alla por el año 2010-2011 se ha generado mucha controversia sobre si el sistema de layouts de Grid reemplazaria a Flexbox, sin embargo la realidad dista mucho de estar en polos opuestos, la verdad es que lo ideal seria usarlos en conjunto para tener una mayor gama de posibilidades y facilidades al momento de diseñar tus fronts.


En este curso aprenderas a usar en conjunto estas dos tecnologias para que tus diseños se hagan mucho mas faciles de realizar.
